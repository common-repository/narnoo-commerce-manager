The Narnoo distributor plugin gives a tourism organisation and tourism wholesalers the ability to create an eCommerce reservations website using their their products found on Narnoo.com.

== Description ==

The Narnoo Commerce plugin allows a tourism business, organisation, wholesaler, online booking site to offer live availability and instant bookings through their WordPress website. 
The plugin is integrated into a number of different payment portals such as PayPal, Stripe and eWay.
This plugin requires either the Narnoo Distributor plugin or the Narnoo Operator Plugin. Which plugin is entirely depending on your Narnoo business account.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure either Narnoo Distributor or Narnoo Operator Plugins have been installed
